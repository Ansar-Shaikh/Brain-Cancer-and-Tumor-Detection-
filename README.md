# 🦴 Bone Cancer Detection & Diagnosis System

An AI-powered web-based system for detecting and classifying bone tumors from medical images (X-ray/MRI). The project focuses on improving early diagnosis, especially in resource-limited healthcare environments.

---

## 🚀 Features
- 🔍 Bone tumor detection using Deep Learning (CNN)
- 📊 Multi-class classification (tumor type & severity)
- 🧠 AI-generated medical summaries using LLM (LLaMA 3)
- 📄 Automated diagnostic report generation
- 🌐 User-friendly web interface for image upload and results

---

## 🛠️ Tech Stack
- **Frontend:** React.js  
- **Backend:** Python (Flask)  
- **AI/ML:** TensorFlow / PyTorch (CNN Models)  
- **Data Processing:** NumPy, OpenCV  
- **Database:** MongoDB  
- **LLM:** LLaMA 3 (for medical explanations)  
- **Deployment:** Docker, Netlify / Render  

---

## ⚙️ How It Works
1. User uploads an X-ray or MRI scan  
2. Image is preprocessed for quality improvement  
3. CNN model analyzes and classifies the tumor  
4. AI generates insights and medical summary  
5. System provides results along with a downloadable report
